# Getting Started
Install packages. In the project directory run:

`yarn install`

# Run tests
Set env. variable so cosole log output is printed:

`set ELECTRON_ENABLE_LOGGING=1`

In Linux:

`export ELECTRON_ENABLE_LOGGING=1`

To run tests in CLI:

`node_modules\.bin\cypress run`
